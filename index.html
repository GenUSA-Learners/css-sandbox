Cascading Style Sheets

Quick Review:

Use an ELEMENT SELECTOR to set all the text inside of each “p” tag to be italic.

Use a CLASS SELECTOR to set all the elements with this class name to be green.

Use an ID SELECTOR to set the specific element with this id to have a colored background.
